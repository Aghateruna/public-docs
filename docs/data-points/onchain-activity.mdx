
# Onchain Activity


| Data Point Name | Description | Data Point Category |
|-----------------|-------------|---------------------|
| Contracts Deployed (Mainnet) | Verifies the total number of smart contracts deployed to mainnet across all supported chains. | <span className="badge badge--primary">Metrics</span> |
| Contracts Deployed (Testnet) | Verifies the total number of smart contracts deployed to testnet across all supported chains. | <span className="badge badge--primary">Metrics</span> |
| Outgoing Transactions | Verifies the total number of outgoing transactions across all supported chains. | <span className="badge badge--primary">Metrics</span> |
| First Transaction | Verifies the timestamp of the first transaction across all supported chains. | <span className="badge badge--primary">Achievements</span> |
| Active Smart Contracts | Verifies the number of smart contracts, deployed to mainnet on all supported chains, with 10+ unique transacting wallets.  | <span className="badge badge--primary">Metrics</span> |
| ETH Balance | Verifies the total ETH balance across all supported chains. | <span className="badge badge--primary">Metrics</span> |
