
# Farcaster


| Data Point Name | Description | Data Point Category |
|-----------------|-------------|---------------------|
| Farcaster Followers | Verifies the number of Farcaster followers. | <span className="badge badge--primary">Metrics</span> |
| Farcaster Account ID | Verifies ownership of a Farcaster account onchain. | <span className="badge badge--primary">Accounts</span> |
| Farcaster Creator Average Score | Checks the average weekly engagement score in Farcaster's rewards system for the top accounts each week. | <span className="badge badge--primary">Metrics</span> |
| Farcaster Creator Rewards | Checks the total amount of USDC earned in Farcaster's rewards system for the top accounts each week. | <span className="badge badge--primary">Affiliations</span> |
| Farcaster Account Age | Verifies the Farcaster account creation date. | <span className="badge badge--primary">Achievements</span> |
| Farcaster Developer Rewards | Checks the total amount of USDC earned in Farcaster's rewards system for the top 25 mini apps each week. | <span className="badge badge--primary">Metrics</span> |
| Farcon NYC 2025 Attendee | Proves attendance at Farcon NYC 2025 in May, by verifying ownership of the attendee ticket NFT. | <span className="badge badge--primary">Achievements</span> |
| Farcaster Spam Label | Checks a Farcaster account's spam classification status, where 0 indicates a high likelihood and 2 indicates a low likelihood. | <span className="badge badge--primary">Metrics</span> |
