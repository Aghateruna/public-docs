
# GitHub


| Data Point Name | Description | Data Point Category |
|-----------------|-------------|---------------------|
| GitHub Crypto Repositories Contributed | Verifies the number of crypto repositories this user has contributed to, based on a curated list. | <span className="badge badge--primary">Metrics</span> |
| GitHub Followers | Verifies the number of GitHub followers. | <span className="badge badge--primary">Affiliations</span> |
| GitHub Repositories | Verifies the number of GitHub repositories this user contributed to over time. | <span className="badge badge--primary">Metrics</span> |
| GitHub Account | Verifies ownership of a Github account. | <span className="badge badge--primary">Accounts</span> |
| GitHub Forks | Verifies the number of repository forks. | <span className="badge badge--primary">Affiliations</span> |
| GitHub Total Contributions | Verifies the total number of GitHub contributions over time. | <span className="badge badge--primary">Metrics</span> |
| GitHub Crypto Repositories Commits | Verifies the number of GitHub commits made to a curated list of crypto repositories. | <span className="badge badge--primary">Metrics</span> |
| GitHub Account Age | Verifies the GitHub account creation date. | <span className="badge badge--primary">Achievements</span> |
| GitHub Stars | Verifies the number of repository stars. | <span className="badge badge--primary">Affiliations</span> |
